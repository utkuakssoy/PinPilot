# PinPilot Privacy Policy

Last updated: June 22, 2026

PinPilot is a local desktop application for Etsy sellers. It helps users import Etsy shop listings, generate Pinterest SEO content, prepare pin drafts, and optionally connect a Pinterest account for publishing.

## Data Stored Locally

PinPilot may store the following data on your own computer:

- Etsy shop listing data imported from a shop URL
- Generated SEO titles, descriptions, keywords, and pin concepts
- Pin drafts and scheduling information
- Pinterest connection tokens, if you choose to connect Pinterest
- Pinterest app credentials, if you choose to save them in the app

This local data is stored in the app's local project data files and is not sold or shared by PinPilot.

## Third-Party Services

PinPilot may connect to third-party services only when you choose to use those features:

- Etsy, to read public shop listing information or future Etsy API data
- Pinterest, to connect your account, read boards, and publish pins
- Google Gemini or another AI provider, to generate SEO content when an API key is configured

Data sent to those services is processed according to their own privacy policies and terms.

## No Sale of Data

PinPilot does not sell user data.

## User Control

Because PinPilot is a local desktop app, you can remove local data by deleting the local data files created by the app.
